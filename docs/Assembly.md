# Assembly Guide

## Required Tools
- Soldering iron (fine tip)
- Tweezers
- Flux
- Multimeter

## Assembly Steps

1. Solder R1 (0603 resistor)
2. Solder D1 (LED)
   - Pay attention to polarity (Cathode marking)
3. Solder TP1 (Test Point Pad)
4. (Optional) Solder J1 (Power Header)

## Visual Check
- No solder bridges
- LED orientation correct
- Test point isolated from ground except via R1 + D1 path
